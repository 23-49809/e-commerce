# Christmas Decor Store

A fully responsive Christmas-themed e-commerce frontend built using HTML, CSS, JavaScript, and Bootstrap 5.  
This project features multiple pages, product listings, category filtering, a cart sidebar, and a festive UI designed for a smooth shopping experience.

## Features

### Frontend Pages
- Home page with hero section and featured products  
- Shop page with:
  - Category filtering
  - Pagination
  - Dynamic product listing  
- About page  
- Contact page with store information and hours  
- Registration/Login UI (frontend only)

### Shopping Cart
- Add to cart  
- Remove items  
- Update quantity  
- Cart sidebar with total price calculation  

### Design
- Responsive layout  
- Bootstrap 5 components  
- Google Fonts and Bootstrap Icons  
- Clean, modern, Christmas-themed UI
